The Moviestore

GitHub Repo

This application was made as part of learning how to be a developer, it was created to showcase the use of Python and flask (Data Centric).
The user can search for a movie using the search bar and the application uses the OMdb API to return information about that movie, users can then leave reviews of that movie and read the reviews left by others. At this time I had not learned about user logins and could not provide functionality for that but the functions and page rendering is all handled by Python which is a back end language.
